# CalculatorHTML
Just my Beta version of calculator. I have python version and HTML
By Kry1zz
сначала вводится первое число
затем вводится число
потом выбирается операция для этого числа
можно повторять сколько угодно
вместо числа можно ввести = для получения результата.



First, enter the initial number.
Then, enter a number.
Next, select an operation for that number.
You can repeat this process as many times as you like.
Instead of a number, you can enter `=` to get the result.
